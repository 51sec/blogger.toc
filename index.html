Blogger Table of Contents (blogger.toc)

Automatic creater for Blogger table of contents(TOC). This script creates TOC of your blogger using feed data.

Demo

  • Try this with your blog !!
  • title order, title print, simple style.
  • published order, title print, simple style.
  • published order, label print, simple style.

Usage

Load "blogger.toc.min.js" on Blogger page like following. User setting should be placed before the concreate script file.

<script type="text/javascript">
// User Setting.
var POSTSTOC_SETTINGS = {
    blogURL: 'garafu.blogspot.jp',  // Set your blog url.
    maxResults: 500,                // Set max recieve data. If set 'Infinity', you can recieve all entries.
    orderby: 'updated',             // Set enum ['title', 'published', 'updated'].
    printby: 'title'                // Set enum ['title', 'label']
};
</script>
<link rel="stylesheet" type="text/css" href="http://garafu.github.io/blogger.toc/release/0.0.2/simple.css" />
<script type="text/javascript" src="http://garafu.github.io/blogger.toc/release/0.0.2/blogger.toc.min.js"></script>
